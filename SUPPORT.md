# Support

Use the right channel so the report can be handled safely:

- Bugs and reproducible feature requests: [GitHub issues](https://github.com/AjnasNB/cockroach-crawler/issues)
- Usage questions and design discussion: [GitHub Discussions](https://github.com/AjnasNB/cockroach-crawler/discussions), when enabled
- Vulnerabilities or sensitive reports: [private security advisory](https://github.com/AjnasNB/cockroach-crawler/security/advisories/new)

Include the package version, Node version, operating system, smallest reproducible input, expected behavior, actual error code, and a sanitized command or program. Remove tokens, cookies, internal URLs, private content, and personal data.

The project does not provide guaranteed response times, hosted crawling, credential brokerage, provider account approval, legal permission to crawl a target, or emergency production support.
