# Cockroach Crawler 0.3.0 release reference

The immutable stable release record is maintained in [../RELEASE.md](../RELEASE.md) and the matching [`v0.3.0` GitHub release](https://github.com/AjnasNB/cockroach-crawler/releases/tag/v0.3.0).

Do not maintain a second, drifting release narrative in the launch kit. Before using version-specific copy, verify:

```sh
npm view cockroach-crawler@0.3.0 version gitHead dist.integrity
npm view cockroach-crawler dist-tags
```

Use `0.3.0` from npm `latest`. Older `0.3.0-alpha.*` notes and screenshots are prerelease history, not current launch truth.
